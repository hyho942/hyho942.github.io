Queue 큐 \ (Q > 자료구조)

FIFO(First-In-First-Out) 방식으로 스택과 반대로 먼저 저장된 값이 제일 먼저 나온다. 즉 가장 최근에 저장된 값은 가장 먼저 나오게된다.

EX) 1,2,3 -> 3,2,1

  • Q.empty() -> Boolean

    - 큐가 비어 있을 때에 true를, 아니면 false 반환한다.
  • Q.enqueue -> None

    - 큐의 맨 뒤에 데이터를 쌓는다.
  • Q.dequeue -> data

    - 큐 맨 앞의 데이터를 삭제하면서 반환
  • Q.peek -> data

    - 큐에서 가장 위(앞)에 있는 데이터을 반환한다.
  • add(item): item을 리스트의 끝부분에 추가한다.
  • remove(): 리스트의 첫 번째 항목을 제거한다.

Linked list 연결 리스트

노드를 단위를 한다. 노드는 데이터와 다음 노드를 가리키는 참조값으로 구성되어 있다.

Sigle linked list

= 이전 데이터로 접근을 할 수 없다.

Operations
  1. S.empty() -> Boolean

    - 리스트가 비었다면 true, 아니면 false
  2. S.size() -> None

    - 리스트에 있는 요소 개수
  3. S.add() -> None

    - 노드를 리스트의 맨 앞에 추가
  4. S.search(target) -> Node

    - 리스트에서 target을 찾는다

    - 찾으면 노드를, 못 찾으면 None 반환
  5. S.delete() -> None

    - 맨 앞 노드를 삭제
