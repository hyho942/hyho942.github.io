프로그래머는 함수를 통해서 추상적인

함수 (function routine procedural, sub-routine)
Procedural -programming = 절차 지향

객체(object) => OOP (object oriented programming) 객체지향

함수 (function)
=> function signature or interface
함수 이름, 매개변수, 결과값(반환)

=> 내부 구현 (implantation)

  1. Function name = 함수가 어떤 일을 하는지 명확하게 나타낸다.
  2. 매개 변수, return
    a. 기능 명세 = 어떤 구조, 어떤 기술, 방식으로 설계할지 설계 (인터페이스 설계)

Mantissa 52bit 는 정밀도를 말한다. 52 bit가 넘으면 에러로 정수로 떨어지지 않는다.

Command =>

sys.float_info.mant_dig
53
53이 나오는 이유는 1.000000…앞에 1이 무조건 적으로 등장하기 때문에 1일 생략하여 mantissa에 52bit 에서 1bit를 초과하여 보여준다.
